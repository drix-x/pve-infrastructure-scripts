# PVE-Config-Backup: Отказоустойчивый бэкап конфигурации Proxmox VE  Скрипт для автоматического создания резервных копий критически важных конфигурационных файлов и базы данных кластера (`pmxcfs`) гипервизора Proxmox VE.   Разработан с упором на максимальную надежность и сквозную защиту от повреждения данных при сетевых или системных сбоях.
